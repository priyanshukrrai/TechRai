HOW TO USE
----------
- Open index.html in any web browser (double-click it).
- The site is a single, self-contained file (HTML + CSS + JS).

NOTES
-----
- The logo uses the "Space Grotesk" font loaded from Google Fonts,
  so an internet connection gives the best look. Offline, it falls
  back to a clean default font automatically.
- All "View certificate" buttons link to the live credential pages.
